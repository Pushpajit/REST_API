ALl Routers

  • /chats to find all the chats
  • /chats?sort={feild, -feild} to sort the chats according to the feild in asc and dec order
  • /chats?select={feild1, feild2, feild3, ...} to select the desired feilds
  • /chats?username={name1, name2, name3, ...} to find the chats of those username(Case Sensitive)
  • /chats/:username to find the chats of those username(Case Insensitive)
  • /chats/:id to modify the chat on that specified id
  • /chats/:id to delete the chat on that specified id
